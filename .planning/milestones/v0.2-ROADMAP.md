# Milestone v0.2: PyQt6 UI + RandomAI + AI Interface

## Overview

Delivered human-vs-AI Chinese Chess board (PyQt6) with RandomAI and extensible AI interface. Human plays either Red or Black (randomly assigned), with AI opponent responding via AIPlayer ABC.

## Milestone Summary

**Key Decisions:**
- PyQt6 float coordinates: fillRect uses QRectF, drawLine uses QLineF (05-05)
- Turn-aware piece selection: piece_value * engine.turn > 0 (06-05)
- EngineSnapshot frozen=True for thread-safe state capture (07-01)
- Double-step undo: human+AI move pair undone together (08-02)
- Random side assignment: human_side randomly chosen at new_game() (08-01)

**Issues Resolved:**
- Palace diagonal position: center columns (3-5) vs side columns (06-08)
- Click offset: mapToScene() replaces hardcoded viewport offset (06-05)
- Turn management: engine.turn used for piece selection (06-05)
- Undo board sync: sync_state() replaces reset_to_state() (08-03)
- Continuous undo deadlock: _start_ai_turn() called when AI turn restored (08-04)

**Issues Deferred:**
- Phase 06: 06-03 automated tests deferred
- Phase 05-07: VALIDATION.md not finalized (nyquist_compliant=false)
- Phase 07: Phase-level SUMMARY.md not created
