**Helper function: `_get_termination_reason(env, result) -> str`**

After a game ends, determine WHY it ended by inspecting `env._engine` internals. This is test-only code that does NOT modify production code (per RESEARCH anti-pattern guidance).

Logic:
```python
def _get_termination_reason(env: XiangqiEnv, result: str) -> str:
    if result in ("RED_WINS", "BLACK_WINS"):
        legal = env._engine.legal_moves()
        if len(legal) == 0:
            if env._engine.is_check():
                return "checkmate"
            else:
                return "stalemate"
        # Has legal moves but game ended -> long chase (chaser loses per endgame.py priority)
        return "long_chase"
    else:  # DRAW
        state = env._engine.state
        # Priority order from get_game_result: repetition > 50-move > long_check > long_chase
        hash_counts = Counter(state.zobrist_hash_history)
        if max(hash_counts.values()) >= 3:
            return "repetition_3fold"
        if state.halfmove_clock >= 100:
            return "50_move_rule"
        if env._engine._rep_state.consecutive_check_count >= 4:
            return "long_check"
        return "unknown_draw"
```

**Main test function: `test_selfplay_100_games()`**

Per D-03: one main test function running full 100-game self-play with statistical assertions.

1. Set `np.random.seed(42)` for reproducibility (Claude's discretion).

2. Initialize `stats` dict with these keys, all as empty lists:
   - `game_lengths` (list of int)
   - `results` (list of str)
   - `total_rewards` (list of float)
   - `termination_reasons` (list of str)
   - `per_step_rewards` (list of float, flattened across all games)

3. Start timing: `t_start = time.time()`.

4. Run 100 games in a loop (`for game_idx in range(100)`):
   - Create fresh `env = XiangqiEnv()` each game
   - `obs, info = env.reset(seed=game_idx)`
   - Initialize per-game counters: `steps = 0`, `total_reward = 0.0`
   - Inner game loop:
     ```python
     while True:
         # R7: legal move verification (D-02)
         engine_legal_count = len(env._engine.legal_moves())
         mask_legal_count = int(info["legal_mask"].sum())
         assert engine_legal_count == mask_legal_count, (
             f"Game {game_idx} step {steps}: engine has {engine_legal_count} "
             f"legal moves but mask has {mask_legal_count}"
         )

         # D-01: sample only from legal_mask indices
         mask = info["legal_mask"]
         legal_indices = np.where(mask == 1.0)[0]
         if len(legal_indices) == 0:
             break  # should not happen if terminated correctly
         action = int(np.random.choice(legal_indices))
         obs, reward, terminated, truncated, info = env.step(action)
         total_reward += reward
         per_step_rewards.append(reward)
         steps += 1
         if terminated:
             break
     ```
   - After game loop ends:
     - `result = env._engine.result()` (string: "RED_WINS", "BLACK_WINS", "DRAW")
     - `reason = _get_termination_reason(env, result)`
     - Append to stats: `game_lengths`, `results`, `total_rewards`, `termination_reasons`

5. End timing: `elapsed = time.time() - t_start`.

6. Print statistics via `print()` (per D-04), formatted as:

   ```
   ========== Self-Play E2E Validation (100 games) ==========
   Timing: {elapsed:.1f}s total ({100/elapsed:.1f} games/sec)
   Avg step time: {total_steps_time / total_steps:.4f}s

   Game Length:
     Mean: {mean}, Median: {median}, Min: {min}, Max: {max}, Std: {std}

   Results:
     RED_WINS: {count} ({pct}%)
     BLACK_WINS: {count} ({pct}%)
     DRAW: {count} ({pct}%)

   Termination Reasons:
     checkmate: {count}
     stalemate: {count}
     repetition_3fold: {count}
     50_move_rule: {count}
     long_check: {count}
     long_chase: {count}
     unknown_draw: {count}

   Reward Statistics:
     Mean: {mean:.4f}, Std: {std:.4f}, Min: {min:.4f}, Max: {max:.4f}
   ==========================================================
   ```

   - **No crash / all games terminate:** `assert len(game_lengths) == 100, f"Only {len(game_lengths)} games completed"`
