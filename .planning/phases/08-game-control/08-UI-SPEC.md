## Interaction Contracts

### Toolbar Button States

| Button | Normal State | AI Thinking | Empty Undo Stack |
|--------|--------------|-------------|------------------|
| 新对局 (New Game) | Enabled | Enabled | Enabled |
| 悔棋 (Undo) | Enabled | Disabled | Disabled |

### Button Behavior

**新对局 (New Game)**
- Click: Reset engine, random assign side, clear board selection, update status bar
- Shortcut: Ctrl+N
- Always available per D-06

**悔棋 (Undo)**
- Click: Call engine.undo() twice (double-step per D-02), clear selection, update status bar
- Shortcut: Ctrl+Z
- Disabled during AI thinking per D-04
- Disabled when undo_stack empty per D-05

### Status Bar Format

Format: `"{side_indicator} | {turn_indicator}"`

| Scenario | Display Text |
|----------|--------------|
| Human plays Red, Red's turn | "你执红方 | 红方回合" |
| Human plays Red, AI thinking | "你执红方 | AI 思考中..." |
| Human plays Black, Black's turn | "你执黑方 | 黑方回合" |
| Human plays Black, AI thinking | "你执黑方 | AI 思考中..." |

## State Transitions

### New Game Flow
```
User clicks "新对局"
  -> GameController.new_game()
  -> Random assign _human_side (+1 or -1)
  -> engine.reset()
  -> board.refresh_pieces()
  -> Clear any selection
  -> Update status bar with new side
  -> If human plays Black: trigger AI move immediately
```

### Undo Flow
```
User clicks "悔棋" (enabled only when valid)
  -> GameController.undo()
  -> engine.undo() twice (double-step per D-02)
  -> board.refresh_pieces()
  -> Clear selection
  -> Update status bar
  -> Always returns to human's turn (Red per D-03)
```
