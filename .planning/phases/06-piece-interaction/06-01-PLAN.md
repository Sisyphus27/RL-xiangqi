    Add selection and piece-index methods to QXiangqiBoard:

    - `_select_piece(row: int, col: int) -> None`:
      Call _clear_highlights() first (per D-13)
      Create selection ring: self._selection_ring = self._create_selection_ring(row, col)
      Call _show_legal_moves(row, col)
      Set self._selected = (row, col)

    - `_show_legal_moves(row: int, col: int) -> None`:
      from_sq = rc_to_sq(row, col)
      For each move in self._engine.legal_moves():
        move_from = move & 0x1FF
        if move_from == from_sq:
          to_sq = (move >> 9) & 0x7F
          to_row, to_col = sq_to_rc(to_sq)
          dot = self._create_legal_move_dot(to_row, to_col)
          self._highlight_items.append(dot)

    - `_deselect_piece() -> None`:
      Call _clear_highlights()
      Set self._selected = None

    - Update _load_pieces() to build _piece_index:
      Initialize: self._piece_index = {}
      After creating each PieceItem: self._piece_index[(row, col)] = piece_item
      This enables O(1) lookup via self._piece_index.get((row, col)) (per D-15)
