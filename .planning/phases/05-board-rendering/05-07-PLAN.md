**Root Cause Analysis:**
The current code incorrectly places palaces at columns 0-2 and 6-8 (left and right sides of the board). In Chinese chess, the palace is a 3x3 area in the CENTER of each player's side:
- Top palace (black): columns 3-5, rows 0-2
- Bottom palace (red): columns 3-5, rows 7-9

**Coordinate Math:**
- Column 3 left edge: (3 + 0.6) * cell = 3.6 * cell
- Column 5 right edge: (5 + 0.6) * cell = 5.6 * cell
- Row 0 top edge: (0 + 0.6) * cell = 0.6 * cell
- Row 2 bottom edge: (2 + 0.6) * cell = 2.6 * cell
- Row 7 top edge: (7 + 0.6) * cell = 7.6 * cell
- Row 9 bottom edge: (9 + 0.6) * cell = 9.6 * cell
