```python
from PyQt6.QtWidgets import QGraphicsEllipseItem, QGraphicsScene, QGraphicsView
from PyQt6.QtCore import Qt, QRectF
from PyQt6.QtGui import QPainter, QPen, QBrush, QColor, QFont
from src.xiangqi.engine.types import Piece, ROWS, COLS
from .constants import RED_FILL, BLACK_FILL, PIECE_TEXT_COLOR, PIECE_STROKE_COLOR, CELL_RATIO, PIECE_FONT_RATIO
```

**Class `PieceItem(QGraphicsEllipseItem)`:**
- `__init__(self, row: int, col: int, piece_value: int, cell: float)`:
  - `self._row = row`
  - `self._col = col`
  - `self._piece_value = piece_value`
  - `self._cell = cell`
  - `d = cell * CELL_RATIO  # 0.8 * cell`
  - Call `super().__init__(0, 0, d, d)`  # local rect 0,0 to d,d
  - `self.setPos((col + 0.2) * cell, (row + 0.2) * cell)`  # scene position
  - `self.setZValue(1.0)`
  - Determine fill: `QColor(RED_FILL)` if `piece_value > 0` else `QColor(BLACK_FILL)`
  - `self.setBrush(QBrush(fill_color))`
  - `self.setPen(QPen(QColor(PIECE_STROKE_COLOR), 2.0))`  # white 2px stroke
  - `self._char = str(Piece(piece_value))`  # Chinese character
- `paint(self, painter: QPainter, option, widget=None)`:
  - Call `QGraphicsEllipseItem.paint(self, painter, option, widget)` to draw ellipse
  - Compute `d = self._cell * CELL_RATIO`
  - Compute `font_size = max(int(d * PIECE_FONT_RATIO), 8)`  # minimum 8px
  - Create `QFont("SimSun", font_size)`  # SimSun with fallback
  - `painter.setFont(font)`
  - `painter.setPen(QPen(QColor(PIECE_TEXT_COLOR)))`  # white
  - `painter.drawText(QRectF(0, 0, d, d), Qt.AlignmentFlag.AlignCenter, self._char)`

**Coordinate mapping (from RESEARCH.md):**
- Scene size: `(COLS + 1.2) * cell` wide x `(ROWS + 1.2) * cell` tall
- Grid occupies 9 cols (0.6 to 9.6 cell), 10 rows (0.6 to 10.6 cell)
- Piece ellipse rect: `(col + 0.2, row + 0.2) * cell` with diameter `0.8*cell`
- Piece center: `(col + 0.6, row + 0.6) * cell`
- Engine row 0 = top (black home), row 9 = bottom (red home)
- Red: `piece_value > 0`; Black: `piece_value < 0`

**Font fallback:** Use `QFont("SimSun, Microsoft YaHei, Arial")` — if SimSun is missing Qt uses YaHei or Arial as fallback.

Use conda env `xqrl` for any Python commands.
